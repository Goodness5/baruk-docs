## Troubleshooting

### Common Issues and Solutions

#### Wallet Connection Problems
**Problem**: Can't connect wallet
**Solutions**:
- Ensure wallet is unlocked
- Check if Sei Network is added
- Clear browser cache and cookies
- Try different browser or incognito mode

#### Transaction Failures
**Problem**: Transactions failing
**Solutions**:
- Check sufficient token balance
- Verify gas fees and limits
- Ensure slippage tolerance is appropriate
- Check network congestion

#### AI Trading Issues
**Problem**: AI not executing trades
**Solutions**:
- Verify strategy is enabled
- Check risk parameters
- Ensure sufficient capital
- Review strategy performance

#### Performance Issues
**Problem**: Slow platform performance
**Solutions**:
- Check internet connection
- Clear browser cache
- Try different browser
- Check system resources
